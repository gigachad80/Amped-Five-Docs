### 2.7 SHARPEN MODULE
**Purpose**: Edge and detail enhancement after restoration

#### **Filters & Sub-Filters**:

**Laplacian Sharpening**
- **What it does**: Simple, fast edge enhancement using second derivative
- **When to use**: ONLY on clean images with minimal noise
- **Key parameters**: Strength (0.3-0.8)
- **Situation**: Already-clean image needing quick sharpness boost
- **Example**: Clean document scan - "Laplacian strength 0.5 → Quick text edge enhancement"
- **MAJOR LIMITATION**: Amplifies noise severely
- **When NOT to use**: Noisy footage, compressed video, low-light imagery
- **No fine control**: Single parameter (strength) only - no ability to protect low-contrast areas
- **Best use**: Preview or exceptionally clean images only
- **Professional preference**: Use Unsharp Masking instead for control

**Unsharp Masking**
- **What it does**: Professional controllable edge enhancement with noise protection
- **When to use**: DEFAULT sharpening choice for forensic work
- **Key parameters**: 
  - **Radius** (0.5-2.0): Edge width
    - 0.5 = fine detail (text, small features)
    - 1.0 = general purpose
    - 2.0 = coarse edges (faces, large objects)
  - **Amount** (0.5-1.5): Sharpening intensity
    - 0.5 = subtle
    - 1.0 = moderate
    - 1.5 = aggressive
  - **Threshold** (0-3): Noise protection
    - 0 = sharpen all edges (including noise)
    - 1-2 = typical (protects low-contrast noise)
    - 3 = only high-contrast edges
- **Situation**: Any image requiring professional sharpening after deblur/denoise
- **Example - License plate**: "Unsharp Mask → Radius 1.2 (character edges), Amount 1.3 (strong), Threshold 2 (protect background noise) → Characters sharp, noise not amplified"
- **Example - Facial detail**: "Unsharp Mask → Radius 0.7 (fine facial features), Amount 0.9 (moderate), Threshold 1.5 → Facial edges enhanced naturally"
- **Advantage**: Threshold parameter prevents noise amplification - critical for compressed/noisy footage
- **Workflow position**: Usually LAST enhancement step, after all restoration and tone correction
- **Professional standard**: Gold standard for forensic sharpening

**Adaptive Sharpen**
- **What it does**: Content-aware sharpening with regional variation
- **When to use**: Mixed detail distribution (text + faces, varied edge strengths)
- **Key parameters**: Strength, adaptation sensitivity
- **Situation**: Document with text AND photographic elements requiring different sharpening
- **Example**: ID card enhancement - "Adaptive Sharpen → Text receives strong sharpening, facial photo receives gentler sharpening automatically → Natural appearance preserved"
- **Advantage**: Different strength for different regions, fewer halo artifacts
- **When to use**: Mixed content where Unsharp Masking over-sharpens some areas

**High-Pass Filter**
- **What it does**: Extracts high-frequency detail for layer blending
- **When to use**: RARE - requires manual post-processing compositing
- **Key parameters**: Radius, blend mode
- **Situation**: Advanced manual compositing workflows only
- **Limitation**: Not directly applicable in typical forensic workflow
- **Note**: Rarely used in standard Amped FIVE workflows - requires external compositing
