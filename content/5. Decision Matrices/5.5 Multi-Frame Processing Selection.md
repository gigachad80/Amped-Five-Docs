### 5.5 Multi-Frame Processing Selection

| Scenario | Motion Type | Best Filter | Parameters |
|----------|-------------|-------------|------------|
| Static scene, no motion | None | **Temporal Smoothing** | Size 5-7 frames |
| Static scene with noise | None | **Frame Averaging** | Average all frames |
| Moving subject, static camera | Object motion | **Motion Smoothing** | Size 5, motion tracking |
| Low resolution, multiple frames | Any | **Super Resolution** | 10+ frames, 2× zoom |
| Perspective changes | Angle variation | **Perspective Stabilization** → **Super Resolution** | Register points, merge |
| Complex: motion + perspective | Both | **Perspective Super Resolution** | Combined filter |
